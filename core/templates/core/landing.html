Get automated help for your GitHub project issues

Cinnamon is a GitHub plugin that helps you to fix issues faster. When a new issue appears, our CinnaBot comment on it suggesting the best project developers to work on it, giving links to other similar issues, and even indicating the code files to be changed.

Features

Our CinnaBot comments on issues with useful info:

Who can solve it?

Who is the best developer to solve an issue? Who can give background about it? CinnaBot knows which developers in your team solved issues similar to the new one and will share this with you.

Any related issues?

Is this new issue related to an old one? CinnaBot answer this by linking to similar issues on your GitHub project. With this you get insight on common problems, regressions, and possible solutions.

Where is the code?

On large codebases it might be difficult to find where the bug is or where the code must be changed. CinnaBot comes to rescue by listing which files changed in previous related pull-requests.

How it works

1. New issue

Cinnamon monitors your GitHub project for new issues. When someone adds one, Cinnamon gets its data to match against other issues and pull-requests of your project. Our natural language expert algorithms are perfect for this.

2. Possible solvers

Knowing who made the pull-requests that fixed similar issues, CinnaBot comments telling who on your project team is most qualified to solve this new issue. That's helpful! But CinnaBot does even more...

3. Useful tips

Once someone is allocated to the issue, CinnaBot will comment on it with useful hints. A list of related issues is given, computed with an advanced similarity algorithm. Also, some files where the problem might be are listed.
